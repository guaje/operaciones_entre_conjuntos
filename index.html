# Operaciones
## Unión

Unión: Definición

Sean \(A\) y \(B\) dos conjuntos, el conjunto unión de \(A\) y \(B\) es el formado con los elementos que están en \(A\) o están en \(B\). El conjunto unión es único, se nota \(A \cup B\) y se define formalmente mediante el siguiente predicado:
$$\phi_{A \cup B} (x) \Leftrightarrow x \in (A \cup B) := (x \in A) \lor (x \in B)$$

Utilizando notación por comprensión
$$A \cup B = \{x : (x \in A) \lor (x \in B)\}$$

Unión: Propiedades

Sean \(A\), \(B\) y \(C\) conjuntos.

Propiedad Nombre
\(A \cup B = B \cup A\) Conmutativa
\((A \cup B) \cup C = A \cup (B \cup C)\) Asociativa
\(A \subseteq A \cup B\)
\(B \subseteq A \cup B\) 		Absorción
\(A \cup \emptyset = A\) Identidad
\(A \cup A = A\) Idempotencia

Unión: Propiedades

Sean \(A\), \(B\) y \(C\) conjuntos.

Propiedad Nombre
\(A \subseteq B \Leftrightarrow A \cup B = B\)
## Intersección

Intersección: Definición

Sean \(A\) y \(B\) dos conjuntos, el conjunto intersección de \(A\) y \(B\) es el formado con los elementos que están en \(A\) y están en \(B\). El conjunto intersección es único, se nota \(A \cap B\) y se define formalmente mediante el siguiente predicado:
$$\phi_{A \cap B} (x) \Leftrightarrow x \in (A \cap B) := (x \in A) \land (x \in B)$$

Utilizando notación por comprensión
$$A \cap B = \{x : (x \in A) \land (x \in B)\}$$

Si \(A \cap B = \emptyset\) se dice que los conjuntos \(A\) y \(B\) son disyuntos o disjuntos.

\((A \cap B) \cap C = A \cap (B \cup C)\) Asociativa
\(A \cap B \subseteq A\)
\(A \cap B \subseteq B\) 		Absorción

\(A \subseteq B \Leftrightarrow A \cap B = A\)

\(A \cup (B \cap C) = (A \cup B)\cap (A \cup C)\) 		Distributiva
## Complemento

Complemento: Definición

Sea \(A\) un conjunto y \(U\) un universo para \(A\), el conjunto complemento de \(A\) es el formado con los elementos que están en \(U\) y no están en \(A\). El conjunto complemento no es único ya que depende del universo con respecto al cual se haga la operación. El conjunto complemento se nota \(\bar A\) y se define formalmente mediante el siguiente predicado:
$$\phi_\bar A (x) \Leftrightarrow x \in (\bar A) := (x \in U) \land (x \notin A)$$

Utilizando notación por comprensión
$$\bar A = \{x : (x \in U) \land (x \notin A)\}$$

Complemento: Propiedades

Sean \(A\) y \(B\) conjuntos, \(U\) un universo para \(A\) y \(B\).

Propiedad Nombre
\(A \cup \bar A = U\) Completación
\(A \cap \bar A = \emptyset\) Nilpotencia
\(\bar {\bar A} = A\) Idempotencia
\(\overline {(A \cap B)} = \bar A \cup \bar B\)
\(\overline {(A \cup B)} = \bar A \cap \bar B\) 		De Morgan

Complemento: Propiedades

Propiedad Nombre
\(\bar U = \emptyset\)
\(\bar \emptyset = U\)
\(A \subseteq B \Leftrightarrow \bar B \subseteq \bar A\)
## Diferencia

$$\phi_{A \setminus B} (x) \Leftrightarrow x \in (A \setminus B) := (x \in A) \land (x \notin B)$$

Utilizando notación por comprensión

Sean \(A\) y \(B\) conjuntos, \(U\) un universo para \(A\) y \(B\).

Propiedad Nombre
\(A \setminus \emptyset = A\) Identidad
\(A \setminus A = \emptyset\) Nilpotencia
\(A \setminus B \neq B \setminus A\) No conmutatividad

Diferencia: Propiedades

Sean \(A\) y \(B\) conjuntos, \(U\) un universo para \(A\) y \(B\).

## Diferencia simétrica

Diferencia simétrica: Definición

Sean \(A\) y \(B\) dos conjuntos, el conjunto diferencia simétrica de \(A\) y \(B\) es el formado con los elementos que están en \(A\) o \(B\) pero no en ambos. El conjunto diferencia simétrica es único, se nota \(A \triangle B\) y se define formalmente mediante el siguiente predicado:
$$\phi_{A \triangle B} (x) \Leftrightarrow x \in (A \triangle B) := x \in ((A \cup B) \setminus (A \cap B))$$

Utilizando notación por comprensión
$$A \triangle B = \{x : x \in ((A \cup B) \setminus (A \cap B))\}$$

Sean \(A\), \(B\) y \(C\) conjuntos, \(U\) un universo para \(A\), \(B\) y \(C\).

\(A \triangle B = B \triangle A\) Conmutativa

## Conjunto de partes

Conjunto de partes: Definición

Sea \(A\) un conjunto, el conjunto partes (o potencia o exponencial) de \(A\) es el conjunto de todos los subconjuntos de \(A\). El conjunto de partes es único, se nota \(\wp (A)\) y se define formalmente mediante el siguiente predicado:
$$\phi_{\wp (A)} (X) \Leftrightarrow X \in (\wp (A)) := (X \subseteq A)$$

Utilizando notación por comprensión
$$\wp (A) = \{X : X \subseteq A\}$$

Conjunto de partes: Ejemplo

Para el conjunto \(A = \{"Cara", "Sello"\}\) se tiene que:

$$\wp (A) = \{\emptyset, \{"Cara"\}, \{"Sello"\}, \{"Cara", "Sello"\}\}$$

Conjunto de partes: Ejemplo

Para el conjunto \(A = \{1, 2, 3\}\) se tiene que:

$$\wp (A) = \{\emptyset, \{1\}, \{2\}, \{3\}, \{1, 2\}, \{1, 3\}, \{2, 3\}, \{1, 2, 3\}\}$$

# Producto cartesiano
## Par ordenado

Par ordenado: Definición

Para todo par de objetos \(a\) y \(b\) existe un único objeto llamado par ordenado de \(a\) con \(b\) y notado \((a, b)\), que se define como:
$$(a, b) = \{\{a\}, \{a, b\}\}$$

Al objeto \(a\) se le llama la primera componente del par ordenado, y al objeto \(b\) se le llama la segunda componente del par ordenado.

Par ordenado: Igualdad

La igualdad entre pares ordenados se establece a través de la siguiente equivalencia lógica:
$$((a, b) = (c, d)) \Leftrightarrow ((a = c) \land (b = d))$$

Par ordenado: Desigualdad

Sean \(a\) y \(b\) dos objetos, se tiene que si \(a \neq b\), entonces los pares ordenados \((a, b)\) y \((b, a)\) no son iguales.

Par ordenado: Ejemplos

  • \((1, 0)\)
  • \((0, 1)\)
  • \((-3, 12)\)
  • \((-3, -1)\)
  • \((0.5, 4.8)\)
  • \((-15.6, 2.7)\)
## Producto cartesiano

Producto cartesiano: Definición

El conjunto producto cartesiano de dos conjuntos \(A\) y \(B\), es el conjunto de todas las parejas ordenadas, cuya primer componente es un elemento del conjunto \(A\) y cuya segunda componente es un elemento del conjunto \(B\). El conjunto producto cartesiano es único y se nota \(A \times B\). El conjunto producto cartesiano puede ser definido mediante el siguiente predicado:
$$\phi_{A \times B} ((x, y)) \Leftrightarrow (x, y) \in A \times B := (x \in A) \land (y \in B)$$

Utilizando notación por comprensión
$$A \times B = \{(x, y) : (x \in A) \land (y \in B)\}$$

Producto cartesiano: Ejemplo

Para los conjuntos \(A = \{"Cara", "Sello"\}\) y \(B = \{"COP", "USD"\}\) se tiene que:

$$A \times B = \{("Cara", "COP")), ("Cara", "USD"), $$ $$("Sello", "COP"), ("Sello", "USD")\}$$
$$B \times A = \{("COP", "Cara")), ("COP", "Sello"), $$ $$("USD", "Cara"), ("USD", "Sello")\}$$

Producto cartesiano: Ejemplo

Para los conjuntos \(A = \{J, Q, K\}\) y \(B = \{\color{blue}{\heartsuit}, \color{black}{\spadesuit}\}\) se tiene que:

$$A \times B = \{(J, \color{blue}{\heartsuit}), (J, \color{black}{\spadesuit}), (Q, \color{blue}{\heartsuit}), (Q, \color{black}{\spadesuit}), (K, \color{blue}{\heartsuit}), (K, \color{black}{\spadesuit})\}$$
$$B \times A = \{(\color{blue}{\heartsuit}, J), (\color{blue}{\heartsuit}, Q), (\color{blue}{\heartsuit}, K), (\color{black}{\spadesuit}, J), (\color{black}{\spadesuit}, Q), (\color{black}{\spadesuit}, K)\}$$

## Producto cartesiano generalizado

Producto cartesiano generalizado: Definición

Sean \(A_1, A_2, \dots , A_n\), \(n\) conjuntos, el producto cartesiano generalizado de dichos conjuntos esta definido por las \(n\)-tuplas ordenadas \((a_1, a_2, . . . , a_n)\) con \(a_i \in A_i, \forall i = 1, 2, 3, \dots, n\).

Utilizando notación por comprensión
$$A_1 \times A_2 \times \dots \times A_n = $$ $$\{(a_1, a_2, . . . , a_n) : (a_i \in A_i), \forall i = 1, 2, 3, \dots, n\}$$

Producto cartesiano generalizado: Ejemplo

Para los conjuntos \(A = \{V, F\}\), \(B = \{1, 0\}\) y \(C = \{\color{cyan}{\blacksquare}, \color{magenta}{\blacksquare}, \color{yellow}{\blacksquare}, \color{black}{\blacksquare}\}\) se tiene que:

$$A \times B \times C = \{(V, 1, \color{cyan}{\blacksquare}), (V, 1, \color{magenta}{\blacksquare}), (V, 1, \color{yellow}{\blacksquare}), (V, 1, \color{black}{\blacksquare}), $$ $$(V, 0, \color{cyan}{\blacksquare}), (V, 0, \color{magenta}{\blacksquare}), (V, 0, \color{yellow}{\blacksquare}), (V, 0, \color{black}{\blacksquare}), $$ $$(F, 1, \color{cyan}{\blacksquare}), (F, 1, \color{magenta}{\blacksquare}), (F, 1, \color{yellow}{\blacksquare}), (F, 1, \color{black}{\blacksquare}), $$ $$(F, 0, \color{cyan}{\blacksquare}), (F, 0, \color{magenta}{\blacksquare}), (F, 0, \color{yellow}{\blacksquare}), (F, 0, \color{black}{\blacksquare})\}$$
